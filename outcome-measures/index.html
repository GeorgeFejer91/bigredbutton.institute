Prototype research instrument

Button Presence and Self-Button Proximity Measure

This prototype adapts pictorial overlap logic for the Big Red Button Institute's 5-minute button press studies. It translates two subjective dimensions into a single visual field: how large the button feels in experience, and how close the button feels to the self.

Larger button circles indicate stronger felt presence. Smaller distance and stronger overlap indicate greater self-button proximity.

Pictorial measure

Self and button in a shared reference frame

The diagram keeps the self fixed and lets the button grow and move as slider values change.

Reference line, containment, and overlap are adapted from pictorial overlap measures used to stabilize subjective spatial comparison.
